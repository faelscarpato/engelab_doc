# README - Modulo Calculo Estrutural IA

## Descricao

Modulo educacional para uso de IA como apoio ao estudo, organizacao, revisao e documentacao de calculo estrutural conceitual.

## Conteudos principais

- Guia de uso do modulo
- Aviso tecnico e responsabilidade
- Fluxo de estudo e revisao
- Checklists
- Prompts base
- Prompts por elemento de concreto armado
- Modelos de memorial
- Exemplos aplicados
- Textos comerciais para integracao na biblioteca
- Planilha editavel de dados e revisao

## Como integrar

Coloque a pasta `04_CALCULO_ESTRUTURAL_IA` na raiz da Biblioteca 50 Projetos-Modelo Engenharia + IA, junto das demais pastas da biblioteca.

## Ordem de uso recomendada

1. Ler `00_GUIA_DO_MODULO`.
2. Escolher um prompt em `01_PROMPTS_BASE`.
3. Usar os modelos por elemento em `02_CONCRETO_ARMADO`.
4. Preencher a planilha em `03_PRE_DIMENSIONAMENTO`.
5. Revisar com os prompts de `04_REVISAO_E_CORRECAO_COM_IA`.
6. Gerar memorial usando `05_MODELOS_DE_MEMORIAL`.
7. Consultar exemplos em `06_EXEMPLOS_DE_USO`.
8. Usar textos comerciais de `07_COMERCIAL_E_INTEGRACAO` quando for apresentar para cliente.

## Aviso tecnico obrigatorio

Este material e um projeto-modelo para estudo, aprendizagem, organizacao de raciocinio estrutural, apoio a memoriais conceituais e revisao preliminar com IA. Nao substitui calculo estrutural executivo, software tecnico validado, norma vigente, levantamento de dados reais, revisao de engenheiro habilitado, ART, responsabilidade tecnica ou aprovacao por orgaos competentes.

Todos os valores, secoes, cargas, combinacoes, criterios, armaduras, verificacoes e conclusoes devem ser recalculados, conferidos e validados por profissional legalmente habilitado antes de qualquer uso real, obra, reforma, intervencao ou decisao tecnica.
