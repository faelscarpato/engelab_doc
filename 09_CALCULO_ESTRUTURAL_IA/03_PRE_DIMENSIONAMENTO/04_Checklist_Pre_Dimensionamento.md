# Checklist de Pre-Dimensionamento Conceitual

## Checklist

### Dados iniciais

- [ ] Tipo de edificacao informado
- [ ] Numero de pavimentos informado
- [ ] Planta ou descricao disponivel
- [ ] Alturas e vaos principais informados
- [ ] Sistema estrutural indicado

### Cargas

- [ ] Carga permanente separada
- [ ] Carga variavel separada
- [ ] Paredes e revestimentos considerados como pendencia quando nao informados
- [ ] Unidade revisada
- [ ] Fonte do valor indicada

### Elementos

- [ ] Vigas nomeadas
- [ ] Pilares nomeados
- [ ] Lajes nomeadas
- [ ] Fundacoes tratadas como dependentes do solo
- [ ] Elementos especiais identificados

### Saida

- [ ] Memorial conceitual gerado
- [ ] Tabela de dados preenchida
- [ ] Revisao com IA realizada
- [ ] Pendencias listadas
- [ ] Validacao profissional indicada

## Aviso tecnico obrigatorio

Este material e um projeto-modelo para estudo, aprendizagem, organizacao de raciocinio estrutural, apoio a memoriais conceituais e revisao preliminar com IA. Nao substitui calculo estrutural executivo, software tecnico validado, norma vigente, levantamento de dados reais, revisao de engenheiro habilitado, ART, responsabilidade tecnica ou aprovacao por orgaos competentes.

Todos os valores, secoes, cargas, combinacoes, criterios, armaduras, verificacoes e conclusoes devem ser recalculados, conferidos e validados por profissional legalmente habilitado antes de qualquer uso real, obra, reforma, intervencao ou decisao tecnica.
