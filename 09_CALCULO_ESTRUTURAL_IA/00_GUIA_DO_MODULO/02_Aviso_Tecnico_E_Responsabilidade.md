# Aviso Tecnico e Responsabilidade

## Finalidade permitida

Este modulo foi criado para uso educacional, treinamento, organizacao de estudo, apoio a producao de material tecnico conceitual e revisao preliminar com IA.

Pode ser usado para:

- estudar conceitos estruturais;
- montar roteiros de calculo;
- organizar dados de entrada;
- gerar memoriais conceituais;
- revisar raciocinios em nivel didatico;
- criar checklists de verificacao;
- apontar informacoes ausentes;
- preparar material de aula, estudo ou apresentacao.

## Finalidade nao permitida

Este modulo nao deve ser usado como:

- projeto estrutural executivo;
- calculo estrutural final;
- documento para obra;
- laudo tecnico;
- parecer de seguranca;
- aprovacao de estrutura;
- substituto de engenheiro calculista;
- substituto de software tecnico validado;
- substituto de ART, CREA, norma vigente ou responsabilidade profissional.

## Regra de linguagem

Sempre usar termos como:

- estudo conceitual;
- valor indicativo;
- a validar;
- a confirmar por calculo;
- dependente de norma vigente;
- dependente de levantamento real;
- requer validacao profissional.

Evitar termos como:

- aprovado;
- seguro;
- dimensionado;
- pronto para obra;
- calculo final;
- conforme garantido;
- estrutura validada;
- substitui engenheiro.

## Regra para prompts

Todo prompt do modulo deve obrigar a IA a:

1. separar dados informados de dados ausentes;
2. nao inventar valores tecnicos;
3. indicar pendencias de validacao;
4. nao aprovar calculos;
5. nao emitir conclusao de seguranca;
6. sugerir revisao por profissional habilitado;
7. manter o carater educacional e referencial.

## Aviso tecnico obrigatorio

Este material e um projeto-modelo para estudo, aprendizagem, organizacao de raciocinio estrutural, apoio a memoriais conceituais e revisao preliminar com IA. Nao substitui calculo estrutural executivo, software tecnico validado, norma vigente, levantamento de dados reais, revisao de engenheiro habilitado, ART, responsabilidade tecnica ou aprovacao por orgaos competentes.

Todos os valores, secoes, cargas, combinacoes, criterios, armaduras, verificacoes e conclusoes devem ser recalculados, conferidos e validados por profissional legalmente habilitado antes de qualquer uso real, obra, reforma, intervencao ou decisao tecnica.
