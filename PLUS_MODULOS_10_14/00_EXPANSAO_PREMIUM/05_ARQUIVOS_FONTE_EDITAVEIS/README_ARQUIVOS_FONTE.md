# Arquivos-fonte editáveis

Esta pasta reúne fontes em Markdown, CSV, JSON e SVG para adaptação dos materiais da expansão premium.

## Conteúdos
- `GUIAS_MD/`: versões textuais dos guias diagramados.
- `MODELOS_CSV/`: modelos tabulares principais usados nas planilhas.
- `PRANCHAS_JSON/`: dados estruturados das pranchas.
- `03_PRANCHAS_CONCEITUAIS/SVG/`: fontes vetoriais editáveis das pranchas.

## Limite de uso
Material educacional, preliminar e referencial. Não substitui análise, projeto, orçamento, laudo, vistoria, documentação de SST ou validação por profissional habilitado.
