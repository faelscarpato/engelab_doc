# Prompt - Memorial de Calculo Conceitual

Atue como redator tecnico especializado em engenharia civil e memoriais estruturais conceituais.

Com base nas informacoes abaixo, gere um memorial de calculo estrutural conceitual para estudo.

Informacoes:
[COLE AQUI DADOS, TABELAS, DESCRICAO DA PRANCHA OU RACIOCINIO]

Estruture o memorial com:
1. Identificacao do projeto-modelo
2. Objetivo do estudo
3. Sistema estrutural considerado
4. Premissas adotadas
5. Dados de entrada
6. Sequencia de calculo ou verificacao
7. Tabelas conceituais, se houver dados suficientes
8. Verificacoes necessarias
9. Limitacoes do estudo
10. Pendencias para validacao profissional
11. Aviso tecnico obrigatorio

Regras:
- Nao assuma responsabilidade tecnica.
- Nao afirme que o calculo esta aprovado.
- Nao declare que o material esta pronto para obra.
- Quando faltarem dados, liste como pendencia.
- Use linguagem tecnica, clara e educacional.

## Aviso tecnico obrigatorio

Este material e um projeto-modelo para estudo, aprendizagem, organizacao de raciocinio estrutural, apoio a memoriais conceituais e revisao preliminar com IA. Nao substitui calculo estrutural executivo, software tecnico validado, norma vigente, levantamento de dados reais, revisao de engenheiro habilitado, ART, responsabilidade tecnica ou aprovacao por orgaos competentes.

Todos os valores, secoes, cargas, combinacoes, criterios, armaduras, verificacoes e conclusoes devem ser recalculados, conferidos e validados por profissional legalmente habilitado antes de qualquer uso real, obra, reforma, intervencao ou decisao tecnica.
