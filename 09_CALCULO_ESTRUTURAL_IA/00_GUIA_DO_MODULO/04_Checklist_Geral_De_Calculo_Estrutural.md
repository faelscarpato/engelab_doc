# Checklist Geral de Calculo Estrutural

## Checklist geral de calculo estrutural conceitual

Use este checklist antes de apresentar qualquer estudo produzido com apoio de IA.

### 1. Dados do projeto

- [ ] Uso da edificacao informado
- [ ] Numero de pavimentos informado
- [ ] Planta-base ou descricao dos ambientes informada
- [ ] Vaos principais identificados
- [ ] Sistema estrutural indicado
- [ ] Restricoes arquitetonicas registradas
- [ ] Informacoes ausentes listadas

### 2. Cargas e acoes

- [ ] Cargas permanentes separadas de variaveis
- [ ] Unidades conferidas
- [ ] Valores informados diferenciados de valores supostos
- [ ] Origem dos valores indicada
- [ ] Norma vigente citada como referencia a confirmar
- [ ] Combinacoes nao tratadas como finais sem validacao

### 3. Elementos estruturais

- [ ] Vigas identificadas por nome
- [ ] Pilares identificados por nome
- [ ] Lajes identificadas por nome ou ambiente
- [ ] Fundacoes tratadas como dependentes do solo
- [ ] Secoes marcadas como conceituais quando nao calculadas
- [ ] Armaduras nao apresentadas como finais sem calculo executivo

### 4. Memorial

- [ ] Objetivo do estudo explicado
- [ ] Premissas registradas
- [ ] Dados de entrada separados
- [ ] Sequencia de calculo apresentada
- [ ] Limitacoes declaradas
- [ ] Pendencias listadas
- [ ] Aviso tecnico obrigatorio incluido

### 5. Revisao com IA

- [ ] A IA apontou possiveis inconsistencias
- [ ] A IA nao aprovou o calculo
- [ ] A IA nao afirmou seguranca estrutural
- [ ] A IA indicou necessidade de validacao profissional
- [ ] O resultado foi revisado por pessoa antes de ser apresentado

## Aviso tecnico obrigatorio

Este material e um projeto-modelo para estudo, aprendizagem, organizacao de raciocinio estrutural, apoio a memoriais conceituais e revisao preliminar com IA. Nao substitui calculo estrutural executivo, software tecnico validado, norma vigente, levantamento de dados reais, revisao de engenheiro habilitado, ART, responsabilidade tecnica ou aprovacao por orgaos competentes.

Todos os valores, secoes, cargas, combinacoes, criterios, armaduras, verificacoes e conclusoes devem ser recalculados, conferidos e validados por profissional legalmente habilitado antes de qualquer uso real, obra, reforma, intervencao ou decisao tecnica.
