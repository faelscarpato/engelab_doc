# MÓDULO IA 15 — Estruturas com IA

**Tema:** vigas, pilares, lajes, fundações, muros, contenções, escadas, reservatórios, estruturas metálicas, madeira, patologias e compatibilização.

> **Aviso de uso:** Este material é educacional/modelo. Não é projeto executivo, laudo, parecer técnico, ART, memorial de cálculo, orçamento oficial, cronograma contratual ou documento legal de segurança do trabalho. Qualquer aplicação real em obra exige levantamento de dados, sondagem quando aplicável, análise estrutural, compatibilização, normas vigentes e validação por profissional habilitado.


## Como usar este ZIP

1. Comece pelo arquivo `01_GUIA_MESTRE/Guia_Mestre_Modulo_IA_15_Estruturas.md`.
2. Use `02_PROMPTS/00_GLOBAL/Prompt_Base_Estruturas.txt` como padrão de geração.
3. Use os prompts individuais EST-01 a EST-50 para criar as pranchas visuais posteriormente.
4. Preencha a planilha em `03_PLANILHAS/Planilhas_Editaveis_Modulo_IA_15_Estruturas.xlsx`.
5. Use os textos comerciais em `05_TEXTOS_COMERCIAIS` para divulgar o módulo.

## Entregáveis incluídos

- Guia mestre do módulo em Markdown.
- Banco global de prompts.
- 50 prompts individuais por projeto-modelo.
- Planilha editável com abas para quantitativos, vistorias, controle de elementos, compatibilização e planejamento visual.
- Modelos editáveis em Markdown.
- Checklists técnicos educacionais.
- Textos comerciais para página de venda, descrição curta e aviso de uso.
- Estrutura de pastas para receber PNG/PDF/DWG/DXF posteriormente.
