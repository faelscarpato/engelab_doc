# Brief de Imagem/Prancha A3 - 05 Edícula com área gourmet

Objetivo: gerar uma prancha técnica referencial para estudo e apresentação.
Descrição: Lançamento estrutural, pilares nos cantos, vigas de cobertura, fundação rasa, área gourmet referencial, detalhes de viga e pilar.
Composição: A3 horizontal, título no topo, diagrama principal central, detalhes laterais, legenda, notas e carimbo simples.
Estilo: AutoCAD/Revit/BIM editorial, fundo branco/cinza claro, linhas grafite, destaques azul técnico.
Formato: 4961 x 3508 px, 300 DPI, textos vetoriais sempre que possível.
Textos visíveis: título, disciplina, legenda, notas técnicas, escala indicada/conceitual, revisão R00, aviso técnico.
Aviso: Projeto-modelo para estudo e referência. Necessita validação por profissional habilitado. Não utilizar como projeto executivo ou documento pronto para obra.
