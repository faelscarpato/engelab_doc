# Exemplo - Residencia Terrea 70m2

## Contexto

Projeto-modelo de residencia terrea de 70 m2 com sala, cozinha, dois quartos, banheiro, area de servico e varanda.

## Uso do modulo

1. Usar prompt de organizacao de dados para listar ambientes, vaos e elementos.
2. Usar prompt de professor para explicar caminho das cargas.
3. Usar prompt de pre-dimensionamento conceitual para montar roteiro.
4. Usar modelo de memorial geral para registrar premissas.
5. Usar prompt de compatibilizacao para comparar prancha e memorial.

## Prompt sugerido

Atue como assistente tecnico educacional. Com base em uma residencia terrea de 70 m2 com sala, cozinha, dois quartos, banheiro, area de servico e varanda, organize um roteiro de estudo estrutural conceitual. Separe dados conhecidos, dados ausentes, elementos estruturais provaveis, cargas a levantar, pendencias e validacoes necessarias. Nao dimensione como projeto final.

## Saida esperada

- Tabela de dados de entrada.
- Lista de vigas, pilares, lajes e fundacoes a estudar.
- Pendencias de levantamento.
- Memorial conceitual.

## Aviso tecnico obrigatorio

Este material e um projeto-modelo para estudo, aprendizagem, organizacao de raciocinio estrutural, apoio a memoriais conceituais e revisao preliminar com IA. Nao substitui calculo estrutural executivo, software tecnico validado, norma vigente, levantamento de dados reais, revisao de engenheiro habilitado, ART, responsabilidade tecnica ou aprovacao por orgaos competentes.

Todos os valores, secoes, cargas, combinacoes, criterios, armaduras, verificacoes e conclusoes devem ser recalculados, conferidos e validados por profissional legalmente habilitado antes de qualquer uso real, obra, reforma, intervencao ou decisao tecnica.
