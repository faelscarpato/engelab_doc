# Como Usar com Com Ia

## Método de uso em 7 passos
1. Defina o objetivo do estudo: compatibilização técnica entre disciplinas de projeto.
2. Separe os documentos de entrada e identifique a fonte de cada dado.
3. Registre o que é dado informado, hipótese, lacuna e pendência.
4. Escolha um checklist ou prompt compatível com a necessidade.
5. Cole o prompt na IA e preencha os campos obrigatórios.
6. Revise a resposta, removendo dados inventados e marcando itens a validar.
7. Encaminhe as pendências para profissional habilitado antes de qualquer uso real.

## Arquivos que você vai usar
| Pasta | Uso |

| 01_GUIA_DO_MODULO | Orientação de uso, fluxo e limites |
| 02_CHECKLISTS | Conferência educacional e preliminar |
| 03_MODELOS_DE_RELATORIO | Documentos editáveis para organizar registros |
| 04_PROMPTS_IA | Prompts prontos para copiar e preencher |
| 05_PRANCHAS_CONCEITUAIS | Briefs visuais e quadros conceituais |
| 06_EXEMPLOS_DE_APLICACAO | Exemplos de preenchimento e uso |
| 07_AVISOS_TECNICOS | Avisos e termos do módulo |

## Resultado esperado
Ao final, o usuário deve obter uma estrutura clara de análise, com evidências, lacunas, perguntas e próximos passos, sem transformar a IA em responsável técnico.

> **Aviso técnico obrigatório:** Este módulo é educacional e preliminar. Não substitui coordenação de projetos, análise BIM, compatibilização executiva, validação normativa, ART/RRT ou decisão de obra. Toda conclusão deve ser a validar por profissional habilitado. Sempre separe dado informado, hipótese, lacuna, pendência e validação profissional.
