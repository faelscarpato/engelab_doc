FAQ

Dúvidas rápidas

Por que alguns agentes aparecem sem link?

Porque o catálogo já inclui agentes presentes no Prompt Master, mas nem todos tiveram URL pública informada. Edite o arquivo script.js e preencha o campo url quando quiser ativar o botão.

A página mostra os prompts internos?

Não. Ela usa os prompts apenas para criar descrições, categorias e entregas. O conteúdo completo dos prompts não fica publicado na interface.

Posso publicar em qualquer hospedagem?

Sim. É uma página estática com HTML, CSS e JavaScript puro. Pode ir para Vercel, Netlify, GitHub Pages, Hostinger ou qualquer hospedagem comum.

Como alterar nomes, categorias e descrições?

Abra script.js, edite o array AGENTS e salve. Os filtros e cards são renderizados automaticamente.
